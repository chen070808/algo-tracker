# 更新公告

## 2026-05-27

这次更新把 OI Life Tracker 从开发模板推进到可发布体验：新用户打开扩展后，不再先看到一长串记录，而是直接看到今天该关注的训练状态。

### 用户体验

- Popup 首屏改为“今日行动面板”：今日提交、AC、连续刷题天数、待复习数、总 Elo、主要短板、近 7 天热力图和下一步建议。
- 最近提交列表从 Popup 移除，避免首屏信息过载。
- 新增 7 天紧凑热力图，适合扩展弹窗内快速判断训练节奏。
- Options 的 About 文案收敛为“刷题能力追踪 + 下一步建议”，不再过早强调 AI 教练。

### 数据可靠性

- 提交记录增加平台和平台题号字段。
- 新增平台 + 题号 + 时间的复合索引，降低重复触发导致的重复记录。
- 导出 JSON 覆盖 submissions、problems、skillProfiles、notes、reviews、achievements。
- 清空本地数据改为二次确认，并清理完整本地状态。

### 发布准备

- README 已替换为产品说明，包含支持平台、安装方式、核心功能和隐私承诺。
- 新增发布检查清单，覆盖 Chrome 商店素材、隐私政策、平台测试矩阵和发布流程。

### 验证

- `npm run build` 已通过。
- 当前仍需继续手动测试洛谷提交检测准确率。
